# Landolt Ring Chart (JIS ratio)

GitHub Pages で動作する静的サイトです。背景色・環の色・環サイズを変更し、視力 0.1〜1.0 のランドルト環を表示します。

## 対応内容

- 視力 0.1〜1.0 を 0.1 刻みで表示
- JIS比率（外径:内径:線幅:切れ目幅 = 5:3:1:1）で描画
- 5 m 視距離・1分角（切れ目）に基づく mm 寸法を算出
- px/mm を入力して画面サイズへ換算（既定値 3.78px/mm）
- 各環の向きを 8 方向でランダム変更（個別/全体）

## 注意

モニターの実寸は機種依存のため、JIS寸法を厳密に再現するには `px/mm` を実測で校正してください。

## ローカル起動

```bash
python3 -m http.server 4173 --bind 0.0.0.0
```

ブラウザで `http://127.0.0.1:4173` を開いて確認できます。
